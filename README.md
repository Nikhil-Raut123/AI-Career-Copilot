# AI Career Copilot

## Overview

AI Career Copilot is an AI-powered career guidance platform built using Python, Streamlit, SQLite, and Gemini AI.

The application helps students and job seekers analyze their skills, evaluate resumes, generate career roadmaps, prepare for interviews, and discover learning resources for different technology career paths.

## Features

* User Authentication (Login & Signup)
* Skill Gap Analysis
* Resume Analyzer (PDF Upload)
* Career Roadmap Generator
* Interview Question Generator
* Career Guidance Chatbot
* Learning Resources
* Certification Suggestions
* Multiple Career Paths Support
* PDF Download for Reports
